# Ecommerce Sales Analysis Dashboard 📊

## Project Overview

This project presents a comprehensive **Ecommerce Sales Dashboard** built using Power BI. It provides deep insights into **sales performance, profitability, and customer behavior** by transforming raw transactional data into meaningful business intelligence.

* Microsoft Power BI Desktop

* Excel (Dataset)

## Dataset

* 🔗 [Details Dataset](https://github.com/AabHi-4321/Ecommerce-Sales-Analysis-Using-PowerbI/blob/main/Ecommerce%20Sales%20Aanlysis_power_bi/Details.xlsx)
* 🔗 [Orders Dataset](https://github.com/AabHi-4321/Ecommerce-Sales-Analysis-Using-PowerbI/blob/main/Ecommerce%20Sales%20Aanlysis_power_bi/Orders.xlsx)

### Dataset Description:

* **Details.xlsx** → Transaction-level data (Amount, Profit, Category, Quantity)
* **Orders.xlsx** → Order-level data (Order Date, Customer Name, State)

---

## 🔄 Data Transformation & Modeling

### ✔ Data Cleaning

* Removed null values
* Adjusted data types
* Eliminated duplicates

### ✔ Data Modeling

* Built relationships between **Orders** and **Details** tables
* Implemented a **star schema structure** for efficient analysis

### ✔ DAX Calculations

Created custom measures:

* Average Order Value (AOV)
* Total Amount
* Total Quantity
* Total Profit

### Product Performance

### Payment Mode Analysis

## Key Insights

*  **Profitability Insight**

  * High sales volume in Clothing
  * Highest profit margins from Printers and Bookcases

*  **Seasonality Trend**

  * Profit peaks: January & November
  * Low performance: May & July

*  **Geographical Insight**

  * Revenue concentrated in a few key states
  * Indicates regional demand concentration
